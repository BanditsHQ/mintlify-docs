Lasso používá kreditový systém pro operace extrakce a obohacení.

## Co spotřebovává kredity

| Operace | Náklady v kreditech |
|---|---|
| **Extrakce tabulky** | Liší se podle velikosti souboru a složitosti |
| **Obohacení jedné buňky** | ~2 kredity za buňku |
| **Hromadné obohacení** | ~0,5 kreditu za sloupec |

Odhady kreditů se vrátí při vytvoření úlohy obohacení, takže znáte náklady ještě před zahájením zpracování.

## Kontrola zůstatku

<CodeGroup>
```typescript TypeScript
const balance = await client.credits.balance();
console.log(`${balance.balance} credits remaining`);
```

```python Python
balance = client.credits.balance()
print(f"{balance['balance']} credits remaining")
```
</CodeGroup>

## Sledování využití

Zobrazte historii kreditových transakcí s volitelnými filtry:

<CodeGroup>
```typescript TypeScript
const usage = await client.credits.usage({
  from: "2025-01-01T00:00:00Z",
  to: "2025-03-31T23:59:59Z",
  service: "enhancement",
});

console.log(`Total used: ${usage.total_used} credits`);
```

```python Python
usage = client.credits.usage(
    from_="2025-01-01T00:00:00Z",
    to="2025-03-31T23:59:59Z",
    service="enhancement",
)

print(f"Total used: {usage['total_used']} credits")
```
</CodeGroup>

## Nedostatečné kredity

Pokud nemáte dostatek kreditů pro operaci, API vrátí chybu `402` s `error_type: "insufficient_credits"`. Zkontrolujte svůj zůstatek před spuštěním rozsáhlých úloh obohacení.

## Předpoklady

- [Vytvořte API key](https://app.lasso.ai) v Lasso dashboardu
- Python 3.8 nebo novější

## Instalace SDK

```bash
pip install lasso-ai
```

## Inicializace klienta

```python
from lasso import LassoClient
import os

client = LassoClient(api_key=os.environ["LASSO_API_KEY"])
```

<Warning>
  Uložte svůj API key do proměnné prostředí. Nikdy ho nezapisujte přímo do zdrojového kódu.
</Warning>

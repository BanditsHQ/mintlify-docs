---
title: 'Získat produkt'
api: 'GET /catalog/{id}'
description: 'Načtení jednoho produktu podle jeho ID'
---

Načte jeden produkt z vašeho katalogu podle jeho jedinečného ID.

<ParamField path="id" type="string" required>
  Jedinečné ID produktu (UUID).
</ParamField>

<ResponseField name="id" type="string">Jedinečné ID produktu.</ResponseField>
<ResponseField name="schema_id" type="string">Schéma katalogu, ke kterému produkt patří.</ResponseField>
<ResponseField name="source" type="string">Zdroj produktu.</ResponseField>
<ResponseField name="status" type="string">Stav produktu.</ResponseField>
<ResponseField name="external_id" type="string">Externí identifikátor (např. Shopify product ID).</ResponseField>
<ResponseField name="attributes" type="object">Atributy produktu jako páry klíč-hodnota.</ResponseField>
<ResponseField name="created_at" type="string">Časové razítko ISO 8601.</ResponseField>
<ResponseField name="updated_at" type="string">Časové razítko ISO 8601.</ResponseField>

<RequestExample>
```typescript TypeScript
const product = await lasso.catalog.get('a1b2c3d4-...');
```

```python Python
product = client.catalog.get("a1b2c3d4-...")
```

```bash cURL
curl -X GET "https://hub.banditshq.com/api/v1/catalog/a1b2c3d4-..." \
  -H "Authorization: Bearer lasso_..."
```
</RequestExample>

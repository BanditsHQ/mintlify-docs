---
title: 'Seznam webhooků'
api: 'GET /catalog/webhooks'
description: 'Výpis všech webhook odběrů pro váš katalog'
---

Načte všechny webhook odběry nakonfigurované pro události katalogu vaší společnosti.

<ResponseField name="webhooks" type="array">
  Pole objektů webhook odběrů.
  <Expandable title="Objekt webhooku">
    <ResponseField name="id" type="string">Jedinečné ID webhooku.</ResponseField>
    <ResponseField name="url" type="string">URL endpointu přijímajícího události.</ResponseField>
    <ResponseField name="events" type="array">Události, na které je webhook přihlášen.</ResponseField>
    <ResponseField name="enabled" type="boolean">Zda je webhook aktivní.</ResponseField>
    <ResponseField name="description" type="string">Volitelný popis.</ResponseField>
    <ResponseField name="created_at" type="string">Časové razítko ISO 8601.</ResponseField>
    <ResponseField name="updated_at" type="string">Časové razítko ISO 8601.</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```typescript TypeScript
const { webhooks } = await lasso.catalog.webhooks.list();
```

```python Python
result = client.catalog.webhooks.list()
```

```bash cURL
curl -X GET "https://hub.banditshq.com/api/v1/catalog/webhooks" \
  -H "Authorization: Bearer lasso_..."
```
</RequestExample>

---
title: 'Bulk delete rows'
description: 'Delete multiple rows from a table at once.'
api: "POST /v1/tables/{table_id}/rows/bulk-delete"
---

## Path parameters

<ParamField path="table_id" type="string" required>
  The unique identifier of the table.
</ParamField>

## Request body

<ParamField body="row_ids" type="string[]" required>
  Array of row IDs to delete. Must be non-empty.
</ParamField>

## Response

<ResponseField name="deleted_count" type="integer">Number of rows successfully deleted.</ResponseField>

<RequestExample>
```typescript TypeScript
const result = await client.tables.bulkDelete("tbl_abc123", {
  row_ids: ["row_1", "row_2", "row_3"],
});

console.log(result.deleted_count); // 3
```

```python Python
result = client.tables.bulk_delete(
    "tbl_abc123",
    row_ids=["row_1", "row_2", "row_3"],
)

print(result["deleted_count"])  # 3
```

```bash cURL
curl -X POST "https://api.lasso.ai/v1/tables/tbl_abc123/rows/bulk-delete" \
  -H "Authorization: Bearer lasso_..." \
  -H "Content-Type: application/json" \
  -d '{
    "row_ids": ["row_1", "row_2", "row_3"]
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "deleted_count": 3
}
```
</ResponseExample>

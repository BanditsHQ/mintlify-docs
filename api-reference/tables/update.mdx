---
title: 'Update table'
api: 'PATCH /v1/tables/{table_id}'
description: 'Update the name, enhancement context, or schema of an existing table.'
---

## Path parameters

<ParamField path="table_id" type="string" required>
  The unique identifier of the table.
</ParamField>

## Request body

All fields are optional. Only provided fields are updated.

<ParamField body="name" type="string">
  New name for the table.
</ParamField>

<ParamField body="enhancement_context" type="string">
  Updated context for AI enhancement operations.
</ParamField>

<ParamField body="schema_id" type="string">
  Assign a different schema to the table. The schema must belong to your company.
</ParamField>

## Response

Returns the full updated table object (same shape as [Get table](/api-reference/tables/get)).

<RequestExample>
```typescript TypeScript
const updated = await client.tables.update("tbl_abc123", {
  name: "Updated Catalog Name",
  enhancement_context: "Focus on technical specifications",
});
```

```python Python
updated = client.tables.update(
    "tbl_abc123",
    name="Updated Catalog Name",
    enhancement_context="Focus on technical specifications",
)
```

curl -X PATCH "https://api.lasso.ai/v1/tables/tbl_abc123" \

</RequestExample>
